# 로또 7 (LOTO 7) 예측 프로그램 상세설계서

## 1. 개요
과거 100회차 당첨 데이터를 기반으로 통계적, 수학적 필터를 적용하여 확률이 높은 로또 7 번호를 조합하는 C# 기반 콘솔 애플리케이션.

## 2. 주요 기능 및 필터 (조합 조건)
1. **[Combo_1_2_3]** 총합, 고저 비율, 홀짝 비율을 모두 만족하는 조합.
2. **[Combo_3_4]** 홀짝 비율을 만족하며, 최근 100회 출현 빈도 기반 가중치를 적용하여 난수를 생성.
3. **[Combo_4_5_6]** 가중치 적용, 연속수(1~2쌍) 포함, 이전 숫자에 따른 마르코프 체인 연관수 기반 생성.
4. **[Combo_1_5_6]** 총합 범위 만족, 연속수 포함, 마르코프 체인 적용.
5. **[Combo_2_6]** 고저 비율 만족, 마르코프 체인 적용.

## 3. 알고리즘 설계
- **총합(Sum) 필터:** 7개 숫자의 합이 100 ~ 160 사이.
- **고저(High/Low) 필터:** 1~18을 '저', 19~37을 '고'로 나누어 3:4 또는 4:3 비율.
- **홀짝(Odd/Even) 필터:** 홀수와 짝수의 비율이 3:4 또는 4:3.
- **연속수(Consecutive) 필터:** 연속되는 번호(예: 12, 13)가 1~2쌍 포함.
- **가중치(Weighted):** 최근 100회 출현 빈도에 비례한 가중치 적용 배열.
- **마르코프 체인(Markov Chain):** 과거 데이터에서 특정 숫자 뒤에 자주 나온 숫자를 추적하여 다음 난수 생성에 개입.

## 4. 실행 방법
```bash
# 번호 생성 모드
dotnet run generate

# 검증 모드 (당첨 번호 입력)
dotnet run verify [번호1] [번호2] [번호3] [번호4] [번호5] [번호6] [번호7]
```
