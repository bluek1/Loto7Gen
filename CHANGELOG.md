# 변경 이력 (CHANGELOG)

## [1.1.0] - 2026-03-15

### 🔒 보안 강화 (Security Enhancements)

#### fetch_data.py 대폭 개선
- ✨ 네트워크 에러 시 자동 재시도 로직 추가 (최대 3회)
- ✨ 타임아웃 설정 (10초)
- ✨ 다중 소스 지원 (Primary + Backup URLs)
- ✨ 상세 예외 처리 (Timeout, ConnectionError, HTTPError)
- ✨ 파일/콘솔 동시 로깅 (`fetch_data.log`)
- ✨ 데이터 검증 강화 (범위, 중복, 개수)
- ✨ 자동 백업 생성 (타임스탬프)

#### 새로운 검증 도구 추가
- ✨ **validate_config.py** - config.json 스키마 기반 검증
  - 필수 섹션/필드 존재 확인
  - 타입 및 범위 검증 (min/max)
  - 기본 설정 생성 기능

- ✨ **validate_history.py** - history.json 무결성 검증
  - 각 추첨 데이터 완전성 검사
  - 통계 분석 (출현 빈도, 평균, 홀짝 비율 등)
  - 데이터 품질 보고

### 🚀 자동화 (Automation)

#### run_pipeline.sh 추가
- ✨ 완전 자동화된 5단계 파이프라인
  1. 환경 검증
  2. 데이터 수집
  3. 모델 학습
  4. 번호 생성
  5. 정리 및 보고

- ✨ 옵션 지원
  - `--dry-run`: 테스트 모드
  - `--skip-fetch`: 데이터 수집 건너뛰기
  - `--skip-train`: 학습 건너뛰기

- ✨ 자동 백업 및 복구
  - 모든 중요 파일 수정 전 백업
  - 실패 시 자동 롤백

- ✨ 오래된 백업 자동 정리 (30일 이상)

- ✨ 색상 구분 로그 출력
  - INFO (파란색)
  - SUCCESS (초록색)
  - WARNING (노란색)
  - ERROR (빨간색)

## [1.0.0] - 2026-03-13 (초기 버전)

### ✨ 핵심 기능

- 🎲 5가지 조합 전략으로 번호 생성
  - Combo_1_2_3: 총합 + 고저 + 홀짝
  - Combo_3_4: 홀짝 + 가중치
  - Combo_4_5_6: 가중치 + 연속수 + 마르코프
  - Combo_1_5_6: 총합 + 연속수 + 마르코프
  - Combo_2_6: 고저 + 마르코프

- 🧠 LSTM 신경망 학습
  - MLX 프레임워크 사용
  - ONNX 형식 지원

- 📊 통계적 필터
  - 총합 범위 (100-160)
  - 고저 비율 (3:4 또는 4:3)
  - 홀짝 비율 (3:4 또는 4:3)
  - 연속수 (1-2쌍)

- 📈 가중치 시스템
  - 빈도 기반 가중치
  - 콜드/핫 번호 보정
